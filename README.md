# 📌 TaskFlow – Smart Task Manager

TaskFlow is a simple web-based task manager I built to improve how I organize daily work.
Most basic to-do apps feel limited, so I added features like search, filtering, and task tracking to make it more practical.

# 🚀 Features
Add, delete, and manage tasks easily, 
Mark tasks as completed, 
Search and filter tasks in real-time, 
Set due dates with overdue task highlighting, 
View task statistics (total, completed, pending), 
Dark mode for better usability, 
Data persistence using browser localStorage.

# 🛠️ Tech Stack
HTML, 
CSS, 
JavaScript
